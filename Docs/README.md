# Docs (AdygGIS / Adygyes)

Эта папка содержит всю документацию проекта. Структура приведена к единому виду.

## Быстрый вход
- Core docs: `Docs/Core/`
  - `Implementation_Plan.md`
  - `PRD_adygyes.md`
  - `project_structure.md`
  - `TechStack_Complete_Guide.md`
  - `UI_UX_doc.md`
  - `AppMap_adygyes.md`
  - `Bug_tracking.md`

## Папки
- `Docs/Release/` — публикация/релизы (Google Play, GitHub Actions, privacy/terms)
- `Docs/Optimization/` — оптимизации APK/ресурсов и data optimization
- `Docs/Integrations/` — интеграции (например Supabase)
- `Docs/UI/` — UI подсистемы (например search results panel)
- `Docs/Markers/` — система маркеров и история изменений
- `Docs/Fixes/` — точечные фиксы и технические заметки
