# Политика конфиденциальности AdygGIS

**Последнее обновление:** 25 декабря 2025 г.

## 1. Введение

Настоящая Политика конфиденциальности определяет порядок обработки и защиты персональных данных пользователей мобильного приложения AdygGIS (далее — «Приложение»). Используя Приложение, вы соглашаетесь с условиями обработки данных, изложенными в настоящей Политике.

## 2. Собираемые данные

### 2.1. Данные о местоположении

Приложение запрашивает доступ к данным о вашем местоположении для следующих целей:

- Отображение вашего текущего местоположения на карте;
- Построение маршрутов до достопримечательностей;
- Определение релевантных точек интереса в вашем регионе;
- Навигация и центрирование карты относительно вашей позиции.

Данные о местоположении обрабатываются исключительно локально на вашем устройстве и не передаются на внешние серверы разработчика.

### 2.2. Технические данные устройства

Автоматически собираемая информация включает:

- Модель устройства и версия операционной системы Android;
- Разрешение экрана и параметры дисплея;
- Версия установленного Приложения;
- Данные о производительности и технических ошибках.

Указанные данные используются для оптимизации работы Приложения, диагностики и устранения технических неполадок.

### 2.3. Пользовательский контент

- Список избранных достопримечательностей;
- Пользовательские настройки интерфейса (тема оформления, язык, параметры карты);
- Локальный кэш карт и изображений.

Весь пользовательский контент хранится исключительно в локальной базе данных устройства и не синхронизируется с облачными сервисами.

## 3. Использование данных

Собранные данные используются исключительно для следующих целей:

- Обеспечение функциональности картографических сервисов и навигации;
- Персонализация пользовательского интерфейса в соответствии с настройками;
- Улучшение производительности и стабильности Приложения;
- Диагностика и устранение технических ошибок;
- Оптимизация пользовательского опыта.

Разработчик не использует собранные данные в маркетинговых целях и не передает их третьим лицам для рекламных рассылок.

## 4. Передача данных третьим лицам

### 4.1. Yandex MapKit

Для предоставления картографических услуг Приложение интегрировано с Yandex MapKit. В рамках данной интеграции передаются следующие данные:

- Координаты для отображения карты и построения маршрутов;
- Поисковые запросы достопримечательностей.

Обработка данных осуществляется в соответствии с Политикой конфиденциальности Яндекса: https://yandex.ru/legal/confidential/

### 4.2. Google Play Services

Для определения местоположения используются сервисы геолокации Google Play Services. Обработка данных регулируется Политикой конфиденциальности Google: https://policies.google.com/privacy

### 4.3. Ограничения передачи данных

Разработчик гарантирует, что следующие данные не передаются третьим лицам:

- Персональная информация пользователя;
- Список избранных мест;
- История поисковых запросов;
- Пользовательские настройки и предпочтения;
- Контакты и другие данные, хранящиеся на устройстве.

## 5. Хранение и защита данных

### 5.1. Локальное хранение

Все данные пользователя хранятся локально на устройстве с использованием защищенной базы данных SQLite (Room Database). Доступ к данным имеет исключительно Приложение AdygGIS.

### 5.2. Меры безопасности

- Применяется стандартное шифрование файловой системы Android;
- Отсутствует передача данных на внешние серверы разработчика;
- Не требуется регистрация или создание учетной записи;
- Соблюдаются рекомендации Android Security Best Practices.

### 5.3. Срок хранения данных

Данные хранятся на устройстве до момента удаления Приложения пользователем. При удалении Приложения все локальные данные автоматически удаляются.

## 6. Права пользователей

### 6.1. Управление разрешениями

Вы имеете право в любой момент отозвать разрешение на доступ к данным о местоположении через настройки операционной системы Android:

```
Настройки → Приложения → AdygGIS → Разрешения → Местоположение
```

Обратите внимание: отключение доступа к местоположению ограничит функциональность навигации и отображения вашей позиции на карте.

### 6.2. Удаление данных

- Для удаления избранных мест используйте функционал Приложения;
- Для полного удаления всех данных деинсталлируйте Приложение;
- Очистка кэша доступна через настройки Android.

### 6.3. Доступ к информации

По запросу пользователя разработчик предоставит информацию о характере собираемых данных и целях их использования.

## 7. Использование приложения несовершеннолетними

Приложение предназначено для пользователей в возрасте от 13 лет. Разработчик не осуществляет целенаправленный сбор персональных данных детей младше 13 лет. Родители и законные представители могут управлять разрешениями Приложения через родительский контроль Android.

Разработчик оставляет за собой право вносить изменения в настоящую Политику конфиденциальности. Существенные изменения будут доведены до сведения пользователей посредством обновления Приложения с соответствующим уведомлением.

Дата последнего обновления указана в начале документа. Рекомендуется периодически проверять актуальную версию Политики.
